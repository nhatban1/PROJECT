# Frontend IUH-EduHub / IUH-EduHub Frontend

Đây là ứng dụng Next.js của hệ thống IUH-EduHub. Tài liệu tổng quan song ngữ và hướng dẫn chạy đầy đủ nằm ở [README gốc](../README.md).

This is the Next.js frontend for IUH-EduHub. The bilingual project overview and full setup instructions are in the root README.

## Chạy nhanh / Quick Start

```bash
cd frontend
npm install
npm run dev
```

## Biến môi trường / Environment Variables

```env
NEXT_PUBLIC_API_BASE_URL=http://localhost:5000/api
```

## Scripts / Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
